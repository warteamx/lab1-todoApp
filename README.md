Hello World!

A full-stack application built with Expo and Express.

SERVER API DOCS: http://56.228.14.41/api-docs

## Project Structure

- **Client**: Expo SDK 53 React Native App (iOS, Android, Web)
- **Server**: Express 5 Node.js API Server

## Quick Start

### Client (Mobile/Web App)
```bash
cd client
npm install
npm start
```

### Server (API)
```bash
cd server
npm install
npm run server:dev
```
